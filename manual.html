Early Alzheimers

When you enter the game, you will be given a set of instructions, which you will have to remember. In our example, we will use the instructions: left, right, left, right, right


On the next page, there will be a car, which you will drive using the left and right arrow keys on your keyboard



For us, we will first press the left arrow on our keyboard to get on the left side of the road. Then, we will go on the right side using the right arrow key, then the left, then the right, and finally we will not press anything, because we want to stay on the right.



After this, we are given a recipe that we need to write down for the next stage.


After the recipe page, we have to enter in the amounts of each item we need (which we have written down from the last page). We can also drag and drop the items into the oven. Now we can start the oven using the "Mix" Button

Finally, we can see our results on the last page!
